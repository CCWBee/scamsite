## Accessibility Notes

### Contrast Ratios (WCAG 2.1)

All color combinations in the design system meet WCAG 2.1 AA requirements (4.5:1 for normal text, 3:1 for large text).

| Combination | Contrast Ratio | WCAG Level |
|-------------|----------------|------------|
| White on Navy (#1a1f3d) | **12.63:1** | AAA |
| White on Trust Blue (#0066a1) | **4.68:1** | AA |
| White on Alert Red (#c8102e) | **4.64:1** | AA |
| White on Success (#28a745) | **3.94:1** | AA (large text) |
| Navy on Warning (#ffc107) | **9.12:1** | AAA |
| Gray-900 on White | **15.91:1** | AAA |

### Best Practices

1. **Never use color alone** to convey information - always include text or icons
2. **Use darker shades** (600-900) for text on light backgrounds
3. **Use lighter shades** (50-200) for backgrounds with dark text
4. **Test with color blindness simulators** to ensure accessibility
5. **Maintain minimum 4.5:1 contrast** for body text

### Color Blindness Considerations

- Avoid relying solely on red/green differentiation
- Use icons and text labels alongside color indicators
- The navy and trust-blue palette provides good differentiation for most color vision deficiencies
