# Typography

Typography in ScamAware Jersey uses Arial as the primary font family to match JFSC branding and ensure maximum cross-platform compatibility without web font loading overhead.

---

## Responsive Typography

Typography scales across breakpoints for optimal readability on all devices.

```jsx
// Responsive heading - smaller on mobile, larger on desktop
<h1 className="text-2xl md:text-3xl lg:text-4xl font-bold">
  Responsive Heading
</h1>

// Responsive body text
<p className="text-sm md:text-base lg:text-lg">
  Responsive paragraph text
</p>
```

### Recommended Responsive Scales

| Element | Mobile (< 768px) | Tablet (768px+) | Desktop (1024px+) |
|---------|------------------|-----------------|-------------------|
| Hero | `text-3xl` | `text-4xl` | `text-hero` |
| H1 | `text-2xl` | `text-3xl` | `text-4xl` |
| H2 | `text-xl` | `text-2xl` | `text-3xl` |
| H3 | `text-lg` | `text-xl` | `text-2xl` |
| Body | `text-sm` | `text-base` | `text-base` |

## Accessibility

- **Minimum font size:** 16px for body text ensures readability
- **Line length:** Keep paragraphs to 65-75 characters (use `max-w-prose`)
- **Contrast:** Use `text-gray-900` or `text-navy` on light backgrounds
- **Avoid all caps:** Use sparingly, only for short labels
- **Don't justify text:** Left-align for better readability
